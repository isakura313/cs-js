Function в JavaScript

Функции в JavaScript - основные строительные кирпичики программ.

Их можно понимать как какие-то подподграммы, которые вызываются только в тот момент, когда это это нужно.

Например, вы покупаете блендер. У него есть функция перемолоть лед. Но она не включается сразу, для этого вам нужно нажать на кнопочку. Точно также работают функции в JavaScript, как и в любом другом языке. Они сработают, только когда их вызывают:


         function blender(ice){
          return ice/16;
        }
        blender(20) // возвращает "измельченное" значение

Сейчас мы с вам просто создали функцию. Давайте сейчас я продемонстрирую шаблон, без которого функция у нас не может создаться.

Всего в JavaScript существует где-то три вида функции: Именнованная, Безымянная, и Стрелочная функция. Сейчас у нас будет представлен каркас именно под Именнованную функцию:


         function Name(a,b ...n){
          // тело функции, которое вызовется только потом
        } // закрывающая фигурная скобка, которая показывает, где функция закрывается

Вместо Name вы можете подставить любое имя, например, переменной, которое является валидным в JavaScript.

В качестве аргумента вы можете передать какие-либо данные, которые функция должна обработать - например, входные значения ( как лед является для вас входным значением, если вы используете функцию "молоть лед"

Безыимянная функция

У функции вовсе не обязательно должно быть имя - если вы например, не собираетесь ее особо реиспользовать потом - просто создали такую функцию, использовали, и на этом дело закончилось.

Достаточно распостраненная сфера применения - выполнение какого-либо действия после определенного рода событий у пользователя


      button.onclick = function(){
        alert("О нет, ты опять нажимаешь по кнопкам");
        }
На самом деле ничто не мешало сделать точно такой же пример и с большой, Именнованной функцией, но это уже совсем другая история.
